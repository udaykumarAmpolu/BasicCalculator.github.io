BASIC CALCULATOR

    How to make basic calculator by using html,css,javascript?

  • First take one form tag give it some name="calculator"

  • we need to take one table inside the form

  • The table contains 4 rows each row contains 4 columns, Except the first row(1st row contains only one column)

  • 1st row is for displaying the result

  • Remaining rows are for Buttons inside columns like numbers and symbols like 1,2,3...+,*,- ...so on

  • 1st row inside coloumn:we take input as text box....type="text"

  • after that each coloumn take input as button ....type="button"....give some name="one"...and also give some value

  • For passing inputs into your display we take onclick(it is inbuilt method in J.S)

    • How to give onclick method?

  • Inside input tag onclick="form name.textbox name.value += 'number or symbol'"

  • "+=" is perform concatenation operation with before values present inside yor display or text box

  • eg:"input type="button" name="one" value="1" onclick="calculator.display.value +='1'"

  • we can give Each number and symbol inputs like above one....

  • At equals symbol it can perform some operation like add,sub,mul,div....for that we can use method called eval()

  • eval() method is inbuilt in java script

    • How to give eval() method?

  • eval(form name.textbox name.value)

  • "input type="button" name="equals" value="=" onclick="calculator.display.value =eval(calculator.display.value)"

  • For Clear screen: "input type="button" name="clear" value="clear" onclick="calculator.display.value=''""
